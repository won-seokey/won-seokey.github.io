서버실 온습도를 **실시간 모니터링**하고, 임계치 초과 시 **자동 알림**을 발송하는 대시보드입니다.

## 기능
- 구역별(Zone) 온습도 차트, 최근 24시간 추이
- 임계값 관리(최소/최대), 다중 알림 메일 등록/삭제
- 관리자 보호(패스워드 확인), 로그/상태 메시지 표시

## 아키텍처 개요
- React SPA + Node.js(Express) API
- MariaDB(`probe_data.sensor_data`) 저장
- Nodemailer 사내 SMTP로 알림 메일 발송
